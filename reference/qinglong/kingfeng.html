# 青龙上传CK

原大佬github仓库地址在新窗口打开

# 搭建方法

# 云服务器拉取docker镜像
docker pull ranqi03/kingfeng:latest

# 部署容器 5000端口可自定义
docker run -dit \
   -v $PWD/kingfeng/:/app/config/ \
   -p 5000:80 \
   --name kingfeng \
   --hostname kingfeng \
   ranqi03/kingfeng:latest
